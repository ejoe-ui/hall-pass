# RHS Handbook Hub Prototype

Static website prototype for the Riverdale High School Parent/Student Handbook 2025–2026.

## Files
- `index.html` – main page
- `styles.css` – visual design
- `app.js` – navigation, search, tabs, helper-language toggle
- `data.js` – handbook section data
- `assets/RHS-Parent-Student-Handbook-2025-26.pdf` – original PDF reference

## Important Accuracy Note
The official handbook text is displayed in English and should remain unchanged. The Spanish layer is a convenience/helper translation only. If the district later provides an approved Spanish PDF, replace the Spanish helper layer with the official approved Spanish text and update the disclaimer accordingly.

## New in this version
- English/Español toggle for the helper layer
- Spanish translations for helper explanations, student view, parent/guardian view, key details, categories, tabs, labels, and scenario cards
- Search includes Spanish helper keywords while preserving official English handbook text


## Revision: Popular Topics Scroll + Leave Campus Fix
- Popular Topics now scrolls inside its panel so it does not push the official handbook section too far down.
- The “Can I leave campus?” scenario now opens a dedicated Closed Campus & Off-Campus Permits section instead of the broader Student Services section.
- Official handbook wording remains unchanged; helper content remains separate.

## Revision Notes
- Popular Topics are now question-style answer cards, not just jumps to official sections.
- Clicking a popular topic opens a quick answer first, followed by the relevant official handbook section.
- Added “What do I need to graduate?” as a popular topic.
- Scenario Helpers remain in the left sidebar and are scrollable.

## Revision: Label update
- Renamed the main top panel label from Popular Topics to Popular Scenario Topics.


## Final usability polish

This version adds priority ordering for common student questions, office-clarification badges for items not clearly answered in the handbook, a clearer Original Page View cue, a brief how-to-use card, and a last-updated footer.
